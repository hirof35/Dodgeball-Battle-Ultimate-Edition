Dodgeball Battle: Ultimate Edition疾走感あふれるドッジボール・シューティングゲーム！迫りくる敵の猛攻をかわし、強化されるショットで敵を撃破せよ。概要上空で左右に動く敵を攻撃し、撃破することを目指すアクションゲームです。敵を攻撃し続けると、プレイヤーの機体がパワーアップし、より強力な攻撃が可能になります。操作方法キー / 操作アクションマウス移動プレイヤーの左右移動スペースキーショット（弾）の発射エンターキーゲーム開始 / リトライゲームの特徴動的な難易度上昇: 時間が経過するごとに敵の攻撃速度と移動速度がアップします。パワーアップシステム: 敵のHPを減らすことで、ショットが2連、3方向へと進化します。ハイスコア保存: 記録は highscore.dat に自動保存され、次回起動時に引き継がれます。視覚効果: 命中時の爆発パーティクルや、被弾時の画面シェイク機能を搭載。導入・実行方法Javaの確認: Java Runtime Environment (JRE) 17以上がインストールされていることを確認してください。フォルダ構成:Plaintext.
├── DodgeballGame.jar (または実行クラスファイル)
├── sounds/
│   ├── bgm.wav
│   ├── shot.wav
│   ├── hit.wav
│   └── lose.wav
└── highscore.dat (自動生成)
実行: コマンドプロンプト等で以下のコマンドを入力するか、IDEから実行してください。Bashjava dodgeballGame.DodgeballGame
開発環境言語: Java 17+ライブラリ: Swing, AWT注意事項音源ファイル（soundsフォルダ）がない場合、音は再生されませんがゲーム自体は動作します。ハイスコアをリセットしたい場合は、highscore.dat ファイルを削除してください。Enjoy the Battle!
